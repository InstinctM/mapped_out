# Mapped Out
### Development Server
#### On Linux:
Install the required modules:

    pip install -r requirements.txt
    # or pip3

Run `run-dev.sh`:
    
    ./run-dev.sh

If this fails, try

    sudo chmod u+x run-dev.sh
    ./run-dev.sh

The web server will be running at `http://localhost:8080` and the api server will be at `http://localhost:8000`
